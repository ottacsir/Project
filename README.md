Lu’s Jewellery

Lu’s Jewellery is a front-end web project built as part of the Introduction to Web Development course.
It simulates an online jewellery shop where users can browse products, filter by category, add items to a cart, and complete a demo checkout.
This project is for educational purposes only. No real purchases are made.

Features

1. Product listing with images and prices
2. Category filtering (e.g. rings, necklaces, earrings)
3. Add to cart functionality
4. Persistent cart using localStorage
5. Cart page with remove, clear, and checkout options
6. Custom modal for feedback and confirmations
7. Contact form with client-side validation

Technologies Used

1. HTML5
2. CSS3 (Flexbox, Grid, Media Queries)
3. JavaScript (DOM manipulation, localStorage)

Project Structure

The project is organized into separate folders to keep files clear and easy to maintain. HTML files are used for page structure, CSS for styling, JavaScript for interactivity, and images for visual content.
1. index.html – Home page of the jewellery store
2. shop.html – Displays all jewellery products with category filtering
3. cart.html – Shopping cart page for managing selected items
4. contact.html – Contact page with store information and feedback form
5. style.css – Contains all styling rules for layout, colors, and responsiveness
6. script.js – Handles cart logic, filtering, modals, and local storage
7. images folder – Contains all product images and user interface images
8. README.md – Project description and usage instructions

How to Run the Project

1. Download or clone the repository
2. Open index.html in any modern web browser
3. Navigate through the pages using the menu
No server or installation is required.
